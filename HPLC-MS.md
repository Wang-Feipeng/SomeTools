# 液相色谱基础介绍

## 液相色谱法的定义

* 液相色谱法就是用液体作为流动相的色谱法，**chromatography**。 

## 液相色谱法的分离机理

* 基于混合物中各组分**对两相亲和力的差别**，是利用混合物中各组分在不同的两相中**溶解、分配、吸附**等化学作用性能的差异，当两相作相对运动是，使各组分在两相中反复多次受到上述各作用力而达到相互分离。

|溶解|分配|吸附|
|-|-|-|

* 两相：stationary phase，固定相；mobile phase，流动相/洗脱剂/溶剂。

## 液相色谱法的分类

* 流动相的物态：气相、液相

* 固定相的形态：平面、薄层

* 相对极性：正相色谱（固定相的极性大于流动相）、反相色谱（固定相的极性小于流动相）

* 有机化合物的极性是**分子间作用力（包括静电引力、偶极力、色散力、氢键力等）的综合表现**的描述

|-|-|-|-|-|-|-|-|-|-|-|

## 色谱分离与检测

* 流动相携带样品进入色谱柱（瞬时进样），流动相连续并稳定通过填料层，样品开始形成一个谱带，由于在流动相和固定相之中存在对各组分的吸附竞争，各组分以不同的速度在分离开的谱带中移动，先后流出色谱柱。

* 当各组分离开色谱柱后，即进入检测器。检测器带有可以在流动相背景下检测到每个分开的化合物谱带的流通池。一个合格的检测器有能力辨别化合物的存在，并发送相应的信号到计算机数据站。根据所需分离和分析的化合物的特性和浓度，可以在众多的检测器中选择其一。如：若一个化合物能吸收紫外光，可使用紫外吸收检测器。若化合物发荧光，可使用荧光检测器。若该化合物没有上述性质，可使用更广泛的检测器，如蒸发光散射检测器ELSD。最强大的方式是顺序使用多个检测器。如紫外和/或蒸发光检测器可和质谱仪MS联用来分析色谱分离的组分。

## 识别色谱图

* 色谱图表示发生在高效液相色谱系统中的化学分离。一系列从基线出现的峰以时间为坐标。每个峰代表对不同化合物的检测器反应。越早被分离的组分保留时间越小。

* chromatogram色谱图，是色谱柱流出物通过检测器时，所产生的响应信号对时间的曲线图，其纵坐标为信号强度（响应值），横坐标为时间（时间/分）。V0=无保留化合物的洗脱体积，等于一个柱体积；V1=化合物1的洗脱体积，可按照色谱峰保留时间和实际流速计算而得；V2=化合物2的洗脱体积；W2=化合物的峰宽（以体积计）。

|色谱峰|峰宽|基线|峰高|死体积|洗脱体积|
|-|-|-|-|-|-|

* 分离度：是两峰相对于其峰宽分开地有多远。R=（V2-Va）/\[1/2(W1+W2)]

* 保留因子k'：保留能力的量度。k'=（V1-V0）/V0

* 理论塔板数N：分离效率的量度。理论塔板数或柱效是一个数值，它表示作为保留时间函数的峰展宽的量度。

* 色谱峰对称性的测量方法：USP拖尾因子（Tailing Factor），Tf=（a+b）/2a;不对称因子（assymmetry Factor），As=b/a。a，b分别为色谱峰5%峰高处的中线两侧峰宽。

## 溶剂管理系统（四元泵和二元泵）

* 四元泵，一般为低压混合，汇合点在泵前。所以一般要先经过脱气机，再在比例阀汇合后，再进泵，两泵串联。

* 二元泵，一般为高压混合，汇合点在泵后。流动相分别进入泵体可以经过脱气机。

* 等度洗脱：在这种模式下，流动相无论是纯溶剂或是预混溶剂，在整个操作过程中流动相比例保持不变。

* 梯度洗脱：在这种模式下，流动相的组成在分离过程中是变化的。这种模式对含多种不同色谱极性化合物的样品非常有用，随着分离的进行，流动相的洗脱强度逐渐增强以洗脱出更强保留的样品组分。

> 梯度洗脱又可分为高压梯度系统或低压梯度系统。

>> 低压梯度系统中，流动相先进比例阀汇合，属于低压混合，混合好后再进泵。

* 自动进样器

> Flow Through Needle流路针设计，进样后，流动相一直冲洗过针头，再到色谱柱，进样体积定量靠注射泵或注射器的吸取定量。

> Sample Loop模式，通过定量环进样，一般三种进样模式：

>> Full Loop Mode满定量环进样，一般2-5倍定量环体积，进样体积定量靠定量环体积。

>> Partial Loop Mode部分定量环进样。

>> u-Pickup Mode补偿进样模式，需要溶剂补偿进样。

## 柱温箱管理系统

* 稳定的温度利于保留时间的稳定，也可降低驻留时间。

* 在较高温度下流动相的粘度较低，易于洗脱。

* 在较高温度下，会形成较低色谱柱反压。

* 在高流速下，小体积色谱柱，流动相的驻留时间短。

## 色谱柱

* 色谱柱是一根填满填料的管子，柱管和填料的种类种类繁多。色谱柱的性能主要取决于填料的性质和填充技术。

## 流动相

* 流动相的要求：过滤除掉微粒，一般推荐使用0.45μm或0.25μm滤膜，分为有机滤膜和水相滤膜；

* 纯度的要求：超纯水电阻率不小于18.2兆欧姆（水中离子越少，电阻率越大，越不易导电，水越纯）。有机溶剂选用色谱纯（溶剂中的杂志含量尽可能低）。

* 防止细菌的生长（一般通过调pH）

* 制备完成后，超声脱气。

* 流动相密封避光保存。

## 定量方法

1. 外标定量法

* 配制一系列已知浓度的标样，保证待测样品浓度在**线性范围**内。

* 采集不同浓度标样的色谱图，积分后，按外标法定量计算，建立标准曲线。

* 计算公式

> 校正因子： *RF(Xi)* = 标样浓度C（Xi）/标样响应值R（Xi）

> 未知组分的浓度：Ci=RF（Xi）×样品响应值Ai

> 特点：1. 无需分组份都被检出，洗脱；2. 需要标样；3. 标样及样品测定的条件要一致；4. 进样体积要准确。

2. 内标定量法

* 配制一系列浓度的标样，其中加有内标样（配制储存标样和储存内标样，按要求配制成工作标样，增加溶质的浓度，但内标样浓度不变）。

* 采集不同浓度标样的色谱图。积分后，按内标定量计算，建立标准曲线（标样峰面积/内标样峰面积）。

* 计算公式

> 校正因子：RF（Xi）=内标样响应值R（I.S）/标样响应值R（Xi）×标样浓度C（Xi）

> 未知组分的浓度：Ci=RF（Xi）×样品峰面积Ai/内标峰面积Ai.s

> 特点：1. 无需各组分都被检出、洗脱；2.需要标样，需要内标样；3. 结果与进样体积无关。

* 对内标物的要求

> 1. 化学结构与待测组分相似（同系物、异构体）

> 2. 在样品中不存在

> 3. 不与样品中组分发生任何化学反应

> 4. 保留值与待测组分接近

> 5. 浓度（响应值）与待测组分相当

> 6. 其色谱峰与其他色谱峰分离好


# QTOF质谱仪

## QTOF的基本构造

1. 离子源

> 常压离子源API：电喷雾离子源ESI、常压化学电离源APCI

> 其他离子源：CaptiveSpray喷雾离子源CSI、大气压光喷雾电离源APPI

* ESI离子源适用于具有极性的绝大多数化合物

* APCI或APPI离子源适用于极性较小的小分子化合物

>> ESI离子化

>>> 离子的产生过程：雾化、脱溶剂、库伦爆炸、解吸离子化

>>> MS模式：四极杆只传输离子，不分离离子。离子流在碰撞池低能力碰撞，产生冷却离子。MS/MS模式：四极杆分离离子。离子流在碰撞池高能量碰撞，产生离子碎片。

## QTOF的基本操作

> 开机

>> 1. 检查打开干燥气（Dry gas）和喷雾气（Nebulizer gas）氮气压力0.5-0.6Mpa，流量大于20L/min；碰撞气（Collision gas）高纯氮气或高纯氩气压力0.6Mpa.

>> 2. 检查机械泵泵油的水平线是否在窗口的1/2-2/3之间。

>> 3. 打开计算机。

>> 4. 开启质谱仪主机电源。

>> 5. 启动microTOFcontrol软件，待真空度达到10e-7mbar后，继续下一步操作。重新开机后，待真空度<2e-6后，需要对检测器进行程序升压。

> 方法设置

>> 1. 双击microTOFcontrol图表打开控制软件

>> 2. 调用方法method-Open

>> 3. 仪器质量准确度校正。仪器在操作状态Operation 稳定20-30分钟后即可开始质量准确度校正。

> 测样方式

>> **注射泵直接进样测定**

>>> 对于标准品或相对较纯或混合组分较少并且不含盐的药物样品，如果仅需要进行鉴定，可以采用直接进样方法测定。样品用标准溶剂（50%H2O、50%乙腈或甲醇、0.1%甲酸）溶解或稀释。设置进样泵的流速为120-180微升/小时。

>> **液质联用（LC/MS）测定**

1. 药物类小分子样品：采用常规液相色谱系统（色谱柱2.1×150mm；流速200-300μL/min）。启动Hystar软件，在Hardware Setup选项中选用适合小分子测定的液相色谱仪模块（接口方法）；选用适合测定小分子的质谱方法；选用或设定适合分离药物小分子的液相方法。

2. 混合蛋白酶解样品：采用纳升液相色谱系统（色谱柱75μm×100mm；流速200-300nL/min）。启动Hystar软件，在Hardware Setup选项中选用适合混合蛋白酶解样品测定的液相色谱仪模块（接口方法）；选用适合测定混合蛋白酶解样品的质谱方法；选用或设定适合分离混合蛋白酶解样品的液相方法。

>> **获取质谱图**

>>> 设置离子源参数，在microTOFcontrol软件中的Source界面可以对仪器各项参数进行调整，以得到适宜的质谱图。

>> **采集数据**

>>> 储存图谱：点sample info，指定图谱存储的文件夹和样品图谱的文件名，点击采集按钮；如果是混合样品需经过HPLC进样并梯度分离后，在线进入质谱分析。数据的储存可以在Hystar软件中完成。

>> **数据处理-DataAnalysis**

1. 打开数据处理软件。

2. 打开数据列表。

3. 选定图谱。

4. Spectrum View窗口右键copy to compound spectra

>> **打印结果**

>> **仪器关机**

>>> 点击microTOFcontrol软件左侧shutdown，出现shutdown option窗口。

>>> 选择shutdown（To switch off all voltages）后，点击OK。

>>> 再次点microTOFcontrol软件左侧shutdown，出现shutdown option窗口。

>>> 选择ventvacuum（To switch off all voltage and to vent vacuum system）后，再次点击OK，出现新界面后点击YES。

>>> 开始释放真空（venting process is active），蓝色进度条示意。耐心等待5到10分钟，氮气将充满整个仪器系统，系统完全失去真空。当在蓝色进度条左侧出现venting is complete，表示释放真空过程完全结束。

>>> 关闭质谱仪主机电源，关闭液氮罐增压阀。
